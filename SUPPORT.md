# Support

Use GitHub Discussions or a feature issue for installation, configuration, and
general usage questions. Include the gateway version, Ubuntu/CloudPanel
versions, a redacted `cloudpanel-gateway doctor` result, and the request ID
from any API error.

Do not include bearer tokens, private keys, passwords, database credentials,
or unredacted production logs. Security reports belong in the private process
described by [SECURITY.md](SECURITY.md).
